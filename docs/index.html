Tutorials

For each tutorial there are two versions of the jupyter notebook:
  1. .ipynb (jupyter notebook file): Jupyter notebook that can execute code when you clone the repo and go through the tutorial on your own machine.
  2. .html: Fully rendered version of the jupyter notebook that can be viewed in your browser without running anything yourself. Cannot run code but you can see the visualizations and follow along.
